About SpamShield

SpamShield is an AI-powered spam detection platform that uses Natural Language Processing (NLP) and Machine Learning to classify SMS and email messages as Spam or Ham.

Project Overview

SpamShield is built as a learning project.

It demonstrates Machine Learning and Full-Stack Web Development.

The application allows users to enter a message and receive a spam prediction.

How It Works

  1. User enters a message.
  2. The message is preprocessed.
  3. TF-IDF converts the text into numerical features.
  4. A trained Naive Bayes model predicts whether the message is Spam or Ham.
  5. The prediction and confidence score are displayed.
